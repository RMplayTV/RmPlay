# RM-TV - Addon Kodi Completo

Um addon profissional para **Kodi** com suporte completo a streaming.

## 🎯 Funcionalidades

✅ **📺 TV AO VIVO** - Streams IPTV com M3U  
✅ **🎬 FILMES** - Gerenciador de filmes  
✅ **📺 SÉRIES** - Organização de séries  
✅ **🧸 DESENHOS** - Conteúdo animado  
✅ **🔍 PESQUISA** - Busca integrada  

## 🚀 Como Usar

1. Instale o addon no Kodi
2. Configure M3U nas Configurações
3. Adicione seus vídeos aos arquivos JSON
4. Comece a assistir!
